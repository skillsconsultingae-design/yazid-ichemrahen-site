# Où mettre tes photos

Dépose tes images **dans ce dossier** (`public/images/`) en gardant **exactement
ces noms de fichiers**. Formats acceptés : `.jpg`, `.jpeg`, `.png` ou `.webp`
(garde le même nom, change juste l'extension si besoin — dis-le moi).

Quand c'est fait, préviens-moi : je branche tout dans `src/lib/img.ts` en 1 min
(tant que ce n'est pas branché, le site continue d'utiliser les placeholders
Unsplash, donc rien ne casse).

### Conseils rapides
- **Grandes images** (1600 px de large minimum, idéalement ~2000 px) — elles
  s'affichent en plein écran. Je pourrai les **alléger** ensuite pour le web.
- Pas besoin de les assombrir toi-même : le site ajoute déjà l'ambiance sombre.
  Mais si tu veux, je peux les **réchauffer / unifier** façon bougie une fois
  déposées.
